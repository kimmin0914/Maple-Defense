# 🍁 메이플 타자 디펜스 (Maple Typing Defense - Web Edition)

추억의 메이플스토리 감성을 담은 **실시간 웹 기반 타자 디펜스 게임**입니다. 
하늘에서 쏟아지는 몬스터와 보스들의 이름을 빠르게 타이핑하여 헤네시스의 평화를 지켜내세요!

### 🚀 [👉 웹에서 바로 플레이하기 (Click to Play!) 👈](https://kimmin0914.github.io/Maple-Defense/)

---

## 🎮 게임 특징 (Features)

* **다양한 몬스터 도감 (50종):** 슬라임, 주황버섯 같은 초보자 몬스터부터 루시드, 검은마법사 등 군단장과 무릉도원의 요괴선사까지 50종의 단어가 이모지와 함께 등장합니다.
* **다이나믹 레벨링 시스템:** 경험치(점수)를 100점 획득할 때마다 레벨이 오르며, 몬스터의 낙하 속도가 점진적으로 빨라져 극강의 텐션을 제공합니다.
* **스마트 스폰 (중복 방지):** 화면에 동일한 단어가 동시에 등장하지 않도록 `do-while` 알고리즘을 적용하여 타자 입력의 억울함을 방지했습니다.
* **HTML5 Canvas 렌더링:** 외부 이미지 파일 없이 순수 JavaScript와 Canvas API만을 사용하여 헤네시스의 주황버섯 집, 흘러가는 구름, 잔디밭 등 맵의 디테일을 직접 렌더링합니다.
* **완벽한 한글 입력 지원:** 숨겨진 `<input>` 태그 포커싱 트릭을 활용하여, 복잡한 한글 조합 과정을 OS에 위임하고 완벽한 한글 타자 경험을 구현했습니다.

## 🕹️ 조작 방법 및 규칙 (How to Play)

1. **시작:** 링크에 접속한 뒤, 화면에서 아무 키나 누르면 즉시 게임이 시작됩니다.
2. **공격 (타이핑):** 떨어지는 몬스터의 이름을 입력한 뒤 `Enter` 키를 누르면 적이 파괴되고 경험치(10점)를 획득합니다.
3. **방어 실패:** 몬스터가 화면 하단의 잔디밭(안내선)에 닿으면 남은 HP가 1 감소합니다.
4. **게임 오버:** HP가 0이 되면 비석이 떨어지며 게임이 종료됩니다. `F5` 키를 눌러 부활(새로고침)할 수 있습니다.

## 💻 로컬 실행 방법 (How to Run Locally)

본 게임은 서버나 별도의 설치 환경이 필요 없는 Vanilla Web 프로젝트입니다.
1. 이 저장소를 `git clone` 하거나 `.zip` 파일로 다운로드합니다.
2. `game.html` 파일을 Chrome, Edge 등 **모던 웹 브라우저**에서 더블 클릭하여 실행합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Language:** HTML5, CSS3, Vanilla JavaScript
* **Rendering:** HTML5 `<canvas>` API (2D Context)
* **Font:** Google Fonts (Jua)
